# NICE NG106 — Chronic heart failure in adults

**Source** : https://www.nice.org.uk/guidance/ng106/chapter/Recommendations
**Version** : 2018 (updated)

---

## Diagnostic — BNP / NT-proBNP

### NT-proBNP seuils
- **>2000 ng/L (236 pmol/L)** : référence urgente, écho < 2 semaines
- **400-2000 ng/L (47-236 pmol/L)** : référence, écho < 6 semaines
- **<400 ng/L (<47 pmol/L)** : IC peu probable (patient non traité)

### Facteurs DIMINUANT les peptides
Obésité, ethnie afro-caribéenne, diurétiques, IEC, ARNI, ARA2, BB, MRA.

### Facteurs AUGMENTANT
Maladie pulmonaire, IR, hépatique, sepsis, BPCO, diabète, cirrhose.

## Écho transthoracique
Pour évaluer :
- Valvulopathie
- Fonction VG systolique et diastolique
- Shunt intracardiaque

## Bilan complémentaire
ECG ; RX thorax ; NFS, créat/iono, TSH, BH, EAL, HbA1c, BU ; peak flow ou spiro.

## Classification par fraction d'éjection (FEVG)

| Type | FEVG |
|---|---|
| **ICFEr** (réduite) | ≤ 40 % |
| **ICFEmr** (modérément réduite) | 41-49 % |
| **ICFEp** (préservée) | ≥ 50 % + anomalie structurelle (2+ : OG > 34 ml/m² SR, E/e' > 11, HVG > 12 mm, HTAP > 35 mmHg) |

## 4 piliers — ICFEr

### Standard
**IEC + BB + MRA + SGLT2-i** d'emblée.

### Optimisation
Si max toléré + symptômes persistants → **switch IEC → ARNI** (sacubitril/valsartan).

### Intolérance IEC (hors angio-œdème)
ARNI + BB + MRA + SGLT2-i

### Angio-œdème IEC ou ARNI
BB + MRA + SGLT2-i + ARA2

## ICFEmr (41-49 %)
Considérer IEC + BB + MRA + SGLT2-i.

## ICFEp (≥50 %)
Considérer **MRA + SGLT2-i** (dapagliflozin TA902 / empagliflozin TA929).

## Fer IV
Si Hb < 150 g/L avec :
- TSAT < 20 % OU
- Ferritine < 100 ng/ml

Options : sucrose, carboxymaltose, derisomaltose.

## Autres options ICFEr
- **Ivabradine** (TA267, 2012) : si FC > 70/min en RS, FEVG ≤ 35 %
- **Hydralazine + nitrés** : intolérance IEC/ARNI/ARA2, surtout afro-caribéen NYHA III-IV
- **Digoxine** : sévère persistante malgré 4 piliers

## CI ICFEr
**Éviter** vérapamil, diltiazem, dihydropyridines courte action.

## Surveillance — IEC/ARNI/ARB/MRA
Avant + 1-2 sem post-instauration + 1-2 sem après chaque dose + tous 3-6 mois au max toléré.

**Seuils intervention** :
- Créat ↑ >50% OU K+ >5.5 mmol/L → adapter localement

## Surveillance — BB
ECG 12 dérivations avant. 
**CI** : BAV 2/3 sans pace, FC < 50.

## Ajustements IRC
- eGFR ≤ 45 : dose initiale plus faible, increments moindres
- eGFR < 30 : MDT en lien avec néphro

## Suivi
Min 6 mois si stable. Plus rapproché si récente modification clinique/TT.
- Évaluation : capacité fonctionnelle, hydratation, rythme, cognition, nutrition
- Revue TT + effets indésirables
- Fonction rénale + bilan martial + Hb

## NYHA
Classe I-IV. Considéré pour MDT spécialisée à NYHA III-IV.

## End-of-life
- PAS d'O2 long terme pour IC avancée seule
- Soins palliatifs intégrés
- Discuter désactivation DAI si fin de vie
- Vaccinations grippe (annuel) + pneumocoque (une fois)

## Rééducation cardiaque
Programme personnalisé, adapté, format accessible, composante psycho-éducative.

## Sel / fluide
PAS de restriction routine. Restreindre fluides si dilutional hyponatrémie. Conseiller si excès. Éviter sels potassiques de substitution.
